### Лабараторная работа 4:
## 3. Подсчёт слов в тексте

### Описание задачи

Необходимо реализовать программу, которая анализирует текст и считает, сколько раз встречается каждое слово.

Это одна из самых классических задач на использование словаря.

Реализовать:

- чтение строки или текста
- разбиение текста на слова
- подсчёт количества повторений каждого слова
- вывод результата в формате `слово -> количество`
- Подсчитывать количество уникальных слов

### Принцип работы : 
- Полученный текст сразу преобразовывается в нижний регистр(чтобы не было проблем по типу "Кот" и "кот" - разные слова) и режется по пробелам
- Циклом проходимся по массиву и записываем каждый его элемент в словарь как ключ, а значение это количество повторений этого элемента в массиве
- Проходимся по значениям словаря, если значение =1 то ключ уникален, счетчик +1
